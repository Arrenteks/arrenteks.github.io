Description

Premise

Gladdus the Gladiator is a Rogue-like Gladiator Arcade Game, developed for the Arcade Machine of the Hochschule Harz. The player embraces the Character of Gladdus and must fight many different Bosses and work with different upgrade and weapon combinations to become the top of the Leaderboard.

Genre(s)

Arcade
Rouge-like
Gladiator Combat

My Contribution(s)

    Programming

  • Designing and Implementation of an Event System based on Unity Scriptable Objects

  • Designing and Implementation of an Upgrade System with different Values for a given Rarity

  • Implementation of a Weapon and Upgrade Selection System with UI

  • Designing and Implementation of a Combo System for different Weapons

  • Designing and Implementation of a Movement

  • Specializing the Combo and Movement System to work on an Arcade Machine/ with and Arcade Stick

  • Implementation of an Highscore System with Leaderboard based on JSON
